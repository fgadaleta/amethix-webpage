installation

Install And Setup In Github

  • 01 Go To Github And Sign in
  • 02 Create A New Repository, for example "jekyll-site"
  • 03 Open The Theme Folder And Unzip it
  • 04 Open CMD (Command Prompot), type "git init" in the command prompot
  • 05 In The Command Prompot, type "git remote add origin < your github repository url >" for example git remote add origin https://github.com/Rubel-hossain/jekyll-site.git
  • 06 In The Command Prompot, type "git add ."
  • 07 In The Command Prompot, type "git commit -m 'init' "
  • 08 In The Command Prompot, type "git push -u origin master"
  • 09 Go To netlify Login With Github
  • 10 Click Add Site Button, Select Your Repository And Click Deploy, Done Your Site Is Live Now

Deployment

To install locally, more to the theme root and run bundle install to install the dependencies, then run jekyll serve to start the jekyll server.

I would recommend checking the Deployment methods page on Jekyll website.
